# chatgpt_api
Testing the openai api

# Instructions
-In VSC, create a .env file in the server folder. Write "OPENAI_API_KEY= " (no quotes). <br />
-In your openai account, generate a new secret key. Paste the key within the env file. <br />
<br />
CLIENT:<br />
cd client <br />
npm run dev <br />
<br />
SERVER:<br />
cd server <br />
node index.js <br />
